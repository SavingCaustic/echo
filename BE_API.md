# BACKEND API DRAFT

## /root/listRacks
Get array of possible racks with name rack(?)

## /rack/setup
Setup new rack
* arg1 = rack number
* arg2 = synth name

## /rack/remove
Removes a rack.

## /synth/changeParam
* arg1 = paramName
* arg2 = paramValue

## /pattern/moveNotes
* arg1 = movement in x direction
* arg2 = movement in y direction
* arg3 = collection of notes
